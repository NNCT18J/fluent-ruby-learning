# fluent-ruby-leaning / 留学生Ruby実習
6/15 〜 10/1でRubyを基礎から学び，プログラミングにより親しむ

# 目的
- Rubyの基本的な文法を習得すること
- Rubyを学習することを通し，プログラミングの考え方を学ぶこと
- 第一言語としてRubyを学習し，次に学ぶ言語の理解につなげること
